# 🌱 Camida Biohosp

**Pour une alimentation saine et 100% bio**

Site web vitrine d'une entreprise spécialisée dans les produits biologiques, notamment les tomates et autres produits agricoles sains.

---

## 📋 Description du Projet

**Camida Biohosp** est une plateforme web présentant une entreprise engagée dans l'agriculture biologique. Le site met en avant :

- Des produits 100% bio et naturels
- Des promotions attractives (ex : "Achetez une tomate, offrez la deuxième")
- Une interface moderne et responsive
- Des sections claires : Accueil, À propos, Produits, Contact et Formation

---

## ✨ Fonctionnalités

- **Header dynamique** avec bandeau défilant (`marquee`)
- Navigation claire entre les pages :
  - Accueil
  - About
  - Produit
  - Contact
  - Formation
- Section **Annonce** avec promotions
- Section **Produits** avec cartes (images + descriptions)
- Bouton "Se Connecter"
- Design responsive
- Utilisation de **Font Awesome** pour les icônes
- Typographie **Poppins**

---

## 🛠️ Technologies Utilisées

- **HTML5**
- **CSS3**
- **Google Fonts** (Poppins)
- **Font Awesome 6.5.1**
- Structure de dossiers propre (`CSS/`, `Html/`, `images/`)
